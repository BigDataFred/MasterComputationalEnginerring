
Ejercicios de la semana 7. La librería no está incluida en este fichero, por
lo que debéis extraer esta carpeta "semana7" en el mismo nivel donde
estuvieran los ejercicios de las semanas anteriores. 

Es decir, la carpeta debe tener la siguiente estructura:

lib/
semana5/
semana6/
semana7/

Los ficheros dentro de la carpeta "semana7/" son los siguientes:

README.txt           Este archivo
ejercicio71.pdf      Documentación del ejercicio 7.1
ej71_texfilter.html  Ejercicio 7.1 (html)
ej71_texfilter.js    Ejercicio 7.1 (js)
ej72_multitex.html   Ejercicio 7.2 (html)
ej72_multitex.js     Ejercicio 7.2 (js)
images/		     Directorio con varias imágenes de texturas
environmentMap.html  Ejemplo de environment mapping (html)
environmentMap.js    Ejemplo de environment mapping (js)
leeperrysmith.html   Ejemplo de aplicación de texturas (html)
leeperrysmith.js     Ejemplo de aplicación de texturas (html)
suzanne.json         El modelo de Suzanne

La documentación del ejercicio 7.2 la tenéis en forma de vídeo.
